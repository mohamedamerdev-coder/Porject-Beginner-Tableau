# Tableau Real Estate Project

## Overview
This project contains interactive Tableau dashboards for analyzing real estate data. The dashboards provide insights into property prices, revenue trends, and area-wise comparisons.

## Dashboards

### Avg price per bedroom
- Shows the average property price for each bedroom count.
- Helps identify pricing trends based on the number of bedrooms.

### Price per zipcode
- Displays property prices grouped by zip code.
- Useful for understanding geographic price distribution.

### Price by zip code
- Compares property prices across different zip codes.
- Highlights areas with higher or lower average prices.

### Revenue for year
- Analyzes total revenue per year.
- Provides insights into annual financial performance.

## Data
- Sample real estate data is used.
- Data includes property prices, zip codes, number of bedrooms, and revenue figures.
